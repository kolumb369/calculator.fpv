🔬 АНАЛІЗ ЧОРНОЇ СКРИНЬКИ

Глибокий аналіз логу польоту. Результати прямо впливають на розрахунок PID: лінійність моторів знижує P при відхиленнях, шумовий поріг D-term визначає агресивність фільтрів.

ПРОГРЕС АНАЛІЗУ
○
2.1 // Валідація логу — перевірка каналів gyro / accel / motor / rpm
○

○
2.3 // Аналіз лінійності моторів: RPM vs Throttle, виявлення десинхрону
○
2.4 // Визначення IR батареї: R = dV / dI при різких стрибках газу
○
2.5 // Профіль струму — крейсерський струм, шумовий поріг D-term
⚠️ Симульовані дані. Аналіз згенеровано автоматично на основі параметрів. Для точного аналізу завантажте реальний Blackbox-лог із Betaflight.
